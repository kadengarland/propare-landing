# Propare Landing Page

Waitlist landing page for [Propare](https://proparesignup.com) — a two-sided coaching marketplace connecting athletes with verified coaches through live 1:1 sessions and video analysis.

## Features
- Athlete + coach waitlist forms via Netlify Forms
- Zapier → Google Sheets integration for both signups
- Scrolling ticker bar and live waitlist counter
- How It Works section with dual phone mockups
- Honeypot spam filter
- Mobile-optimized footer

## Stack
- Single HTML file (Tailwind CSS via CDN)
- Netlify Forms for form handling
- Netlify Drop for deployment

## Deployment
This site deploys manually via **Netlify Drop** (drag-and-drop). Auto-deploy from GitHub is not active due to a deprecated build image error.
